# AudioMod
Web audio player that allows you to upload local audio files and modify playback with audio effects.

## Features
- Playback speed control
- Reverb
- Volume booster
- 6-band parametric equalizer

## Run locally
1. Clone repo
   ```
   git clone https://github.com/hzyfaa/AudioMod
   cd AudioMod
   ```
2. Install packages
   ```
   npm install
   ```
3. Run
   ```
   npm run dev
   ```
